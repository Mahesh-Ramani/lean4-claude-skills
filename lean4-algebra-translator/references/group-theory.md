## Subgroups

```lean
-- The type
H : Subgroup G

-- Key subgroup constructors
Subgroup.closure S        -- smallest subgroup containing S ⊆ G
Subgroup.zpowers a        -- cyclic subgroup ⟨a⟩
Subgroup.center G         -- center Z(G)
Subgroup.normalizer H     -- normalizer of H

-- Membership
x ∈ H                     -- definitionally: H.carrier x
Subgroup.mem_closure      -- x ∈ closure S ↔ ...
Subgroup.mem_zpowers_iff  -- x ∈ zpowers a ↔ ∃ n : ℤ, a ^ n = x

-- Lattice operations
H ⊓ K                     -- intersection
H ⊔ K                     -- join (generated by H ∪ K)
⊤                         -- whole group
⊥                         -- trivial subgroup

-- Index
H.index                   -- [G : H]
Subgroup.index_eq_card    -- H.index = G.card / H.card (finite groups)

-- Normal subgroups
H.Normal               -- H is normal in G
Subgroup.normal_iff_forall_conj  -- ∀ g n, g * n * g⁻¹ ∈ H
```

## Cosets and quotient groups

```lean
-- Left cosets
leftCoset a H             -- a • H
-- Right cosets
rightCoset H a            -- H • a

-- Quotient group (H must be normal)
G ⧸ H                     -- QuotientGroup.quotient H

-- Quotient map
QuotientGroup.mk : G →* G ⧸ H

-- Key facts
QuotientGroup.mk_eq_one    -- mk a = 1 ↔ a ∈ H
QuotientGroup.eq           -- mk a = mk b ↔ b⁻¹ * a ∈ H

-- First Isomorphism Theorem
QuotientGroup.quotientKerEquivRange  -- G/ker(f) ≅ range(f)
```

## Lagrange's theorem

```lean
-- Card of a group equals index * card of subgroup
Subgroup.card_mul_index   -- Fintype.card H * H.index = Fintype.card G

-- Lagrange: order divides
Subgroup.card_dvd_of_le   -- H ≤ K → card H ∣ card K

-- Order of element divides order of group
orderOf_dvd_card          -- orderOf g ∣ Fintype.card G

-- Order of element
orderOf g                 -- ℕ, the multiplicative order of g
orderOf_eq_card_zpowers   -- orderOf g = (zpowers g).card
pow_orderOf_eq_one        -- g ^ orderOf g = 1
```

## Group homomorphisms

```lean
-- Type
f : G →* H              -- MonoidHom G H (the * denotes monoid homomorphism)

-- Kernel and range
f.ker                   -- MonoidHom.ker f : Subgroup G
f.range                 -- MonoidHom.range f : Subgroup H

-- Properties
f.injective             -- f is injective
MonoidHom.injective_iff_ker_eq_bot  -- injective ↔ ker = {1}
f.surjective

-- Isomorphisms
e : G ≃* H              -- MulEquiv G H
e.toMonoidHom           -- underlying hom
```

## Group actions

```lean
[MulAction G X]

-- Orbit and stabilizer

MulAction.stabilizer G x   -- the stabilizer of x (a Subgroup G)

-- Orbit-Stabilizer theorem
MulAction.card_orbit_mul_card_stabilizer_eq_card_group
-- |G·x| * |Stab(x)| = |G|

-- Fixed points
MulAction.fixedPoints G X  -- {x : X | ∀ g, g • x = x}
```

## Sylow theorems

```lean
import Mathlib.GroupTheory.Sylow

variable {G : Type*} [Group G] [Fintype G] {p : ℕ} [Fact (Nat.Prime p)]

-- A Sylow p-subgroup
Sylow p G                  -- the type of Sylow p-subgroups

-- First Sylow theorem: existence
Sylow.nonempty             -- Sylow p G is nonempty

-- Second Sylow theorem: conjugacy
Sylow.conj_eq              -- any two Sylow p-subgroups are conjugate
IsPGroup.sylow_mem_fixedPoints_iff

-- Third Sylow theorem: count
card_sylow_dvd             -- number of Sylow p-subgroups divides |G|
card_sylow_modEq_one       -- number ≡ 1 [MOD p]
```

## Worked examples

### Order of an element divides group order

```lean
example {G : Type*} [Group G] [Fintype G] (g : G) :
    orderOf g ∣ Fintype.card G :=
  orderOf_dvd_card
```

### Normal subgroup check via conjugation

```lean
example {G : Type*} [Group G] {H : Subgroup G}
    (h : ∀ g : G, ∀ n : G, n ∈ H → g * n * g⁻¹ ∈ H) : H.Normal :=
  { conj_mem := h }
```

### Quotient group is abelian when G is abelian

```lean
example {G : Type*} [CommGroup G] {H : Subgroup G} [H.Normal] :
    CommGroup (G ⧸ H) :=
  inferInstance
```

### First isomorphism theorem

```lean
example {G H : Type*} [Group G] [Group H] (f : G →* H) :
    G ⧸ f.ker ≃* f.range :=
  QuotientGroup.quotientKerEquivRange f
```
